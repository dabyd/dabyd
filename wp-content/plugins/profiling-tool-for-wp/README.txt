A plugin for testing the performance of the themes, plugins and scripts of your Wordpress site.

== Description ==

This plugin help you to test and check the performance of the plugins, themes and scripts you have
installed in your wordpress site.

Its bring easy to read tables and charts that show what theme, plugin or page are the ones that are slowing down your website.

Its comes in several languages that guaranteed you will know how to use it or what is the problem in you site.

It makes it very simple to locate any bottleneck problem in your themes or plugins in order to solve it and speed up your website.

Save your scans and profiles in a history table to later use in your performance reports.

Install and active the plugin so you can start checking your wordpress site right away.

= Features =

* Plugins & themes performance profiling.
* Page performance profiling.
* Saved profiling history scans.
* Multi-language (EN, ES, GL).
* Testing single themes or plugins.
* Testing multiple pages charging times at once.

== Frequently Asked Questions ==

= What is Profiling tool for WordPress? =

This is a plugin that helps you measures the performance of your plugins and themes, it helps you in analize the performance of the plugins you have installed in your site and allows you to find if any of them is having any performance problems. It also helps you to test your single wordpress pages to watch their load time and the performance that they are having and all the files they load when they are called by your site allowing you to performe improvents if any of the pages of the site are usually loading slower than others.

= What are the requirements for running Profiling Tool? =

It requires at least WordPress 5.0 and PHP 7.1. .

= Profiling Tool For WP will work if im using any extra cache or optimization plugin? =

It will work with the most of any other active plugin in your site, you can check if another cache plugin is affecting WP Performance if you see the result of the test with similar values that means the plugin test has been affected by others. We encourge you to contact us if you find a plugin that affect the main result of the WP Performance.

= Do I need to deactivate or uninstall Profiling Tool when I'm not using it ? =

You don't need to deactivate or uninstall the plugin if you are not using it, generally it will not affect in any means the rest of the plugins or pages in your site.

= Is it possible to test a single plugin during the profiling? =

Profiling Tool offers the option to test a single plugin to do the test in that plugin in his main table where all the plugins and themes in your site are listed. This will run the test only on that plugin.

= What is your Privacy Policy? =

The plugin does not collect any private data from you or your visitors. It does not use cookies either. Your website can run Profiling Tool and be compliant with general privacy laws.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `profiling-tool-for-wp.zip` to the `/wp-content/plugins/` directory or directly add it from the wordpress plugins section.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the main plugin page throught the new added link in the main wordpress sidebar
